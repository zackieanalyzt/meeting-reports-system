## üîÑ Database Logic

### Meeting Status Calculation

```sql
SELECT 
    m.*,
    (SELECT COUNT(*) FROM meeting_agendas 
     WHERE meeting_number = m.meeting_number) as agenda_count,
    (CASE WHEN m.file_size > 0 THEN true ELSE false END) as has_report,
    (CASE WHEN m.meeting_date > CURRENT_DATE THEN true ELSE false END) as is_upcoming
FROM meeting_reports m
ORDER BY m.meeting_date DESC;
```

### Meetings Without Reports

```sql
SELECT id, meeting_number, meeting_title, meeting_date
FROM meeting_reports
WHERE file_size = 0 OR file_size IS NULL
ORDER BY meeting_date DESC;
```
