
6. คลิก "บันทึก"

## 🐛 แก้ปัญหา

### ปัญหา: ไฟล์ไม่แสดง

**วิธีแก้**:
1. Restart backend
   ```bash
   cd backend
   npm start
   ```
2. Hard refresh browser (Ctrl+Shift+R)
3. ลอง logout แล้ว login ใหม่

### ปัญหา: ใช้งานจากเครื่องอื่นไม่ได้

**วิธีแก้**:
1. ตรวจสอบ IP ของเครื่อง server
   ```bash
   ipconfig  # Windows
   ifconfig  # Mac/Linux
   ```
2. ใช้ IP นั้นในเบราว์เซอร์
   ```
   http://[IP]:3000
   ```
3. ตรวจสอบ Firewall (อาจต้องเปิด port 3000 และ 3001)

### ปัญหา: อัพโหลดไม่ได้

**เช็คก่อน**:
- ✅ ไฟล์ขนาดไม่เกิน 20MB
- ✅ ไฟล์เป็นประเภทที่รองรับ
- ✅ อัพโหลดไม่เกิน 5 ไฟล์
- ✅ Login ด้วย role secretary หรือ manager

**วิธีแก้**:
1. ตรวจสอบขนาดไฟล์
2. ตรวจสอบประเภทไฟล์
3. ลอง logout แล้ว login ใหม่
4. ตรวจสอบ console (F12) ดู error

### ปัญหา: CORS Error

**วิธีแก้**:
1. Restart backend
   ```bash
   cd backend
   npm start
   ```
2. ตรวจสอบว่า backend รันอยู่
   ```bash
   curl http://localhost:3001/api/health
   ```
